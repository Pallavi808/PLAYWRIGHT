# Playwright JavaScript Practice

This repository contains JavaScript practice files for learning basic language concepts.

## Chapters

- `Chapter_01_JS_Basics/` - Introductory JS examples and simple commands.
- `Chapter_02_Js_concepts/` - Identifier rules, comments, and variable naming examples.

## Notes

- `07_Identifier_part2.js` demonstrates different identifier casing styles.
- `08_comments.js` contains examples of JavaScript comments.

## Usage

Open the files in a code editor and run them with Node.js or use them for study and reference.
